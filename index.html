FrogKnight

Frogknight is a colorful action-adventure game still in development in which you play an adorable frog that must free the kingdom from a terrible curse.

The game is being built on top of my own game engine (Corgi Engine) and will feature a whole world for the player to explore, filled with dungeons, treasures and colorful characters to interact with!

Corgi Game Engine

Corgi is a C++/OpenGL game engine I'm currently working on to build my other projects. It focuses on 2D features but still support 3D operations or meshes.

You can find the source code on my github repository

Features :

  • 2D Support (Sprite Animation, Tilemapping, Sorting etc)
  • Basics Physics (2D collisions detection)
  • UI System (Fonts, Widgets)
  • CrossPlatform (Linux/Windows/MacOS)

FrogKnight Game Editor

I'm working on a game editor to make it easier to create levels for my game FrogKnight. The Editor is made using C++ and the GUI features of my engine (corgi engine).

So far, the Editor allow tilemapping edition and the ability to run the game while editing the level in real time
